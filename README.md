<h1 align="center">Welcome to Write &Share 👋</h1>

> A simple blogging site where you can write and share your blogs with everyone! This project was made with the MERN stack, a popular javascript stack used to create full-stack web apps. Not too exciting project just took very long haha

### ✨ [Demo](https://write-share.web.app/)

## Install(in both folders)

```sh
npm install
```

## Run frontend

```sh
npm start
```

## Spin up backend server

```sh
npm run dev
```
